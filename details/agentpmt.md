## Description
AgentPMT is the universal MCP marketplace that gives AI agents instant access to a growing library of tools, workflows, and automations. Rather than coding individual integrations, your agents discover and execute capabilities on demand through a single MCP connection -- from SaaS platform access and API orchestration to complete business process automation.

Agents use credit allocations to purchase and execute tools and workflows on demand, and can even make purchases via x402 stablecoin if authorized. Agents can use encrypted credentials to access software platforms and make credit card purchases without ever having access to the actual credentials. All credit card flows have a human in the loop via the mobile app approval process. All agent interactions are tracked in the central dashboard for easy troubleshooting and prompt engineering flows for end to end visibility.

## Features
- **Universal MCP marketplace** - Single MCP connection to a growing library of tools, workflows, and automations
- **On-demand tool discovery and execution** - Agents discover and execute capabilities without coding individual integrations
- **Credit-based purchasing** - Agents use credit allocations to purchase and execute tools and workflows
- **x402 stablecoin payments** - Optional blockchain-based payments for authorized agents
- **Encrypted credential management** - Agents access software platforms and make credit card purchases without seeing actual credentials
- **Human-in-the-loop approvals** - All credit card flows require approval through the mobile app
- **Central dashboard** - Track all agent interactions for troubleshooting and prompt engineering
- **SaaS platform access** - Connect agents to third-party software platforms
- **API orchestration** - Coordinate multiple API calls into unified workflows
- **Business process automation** - End-to-end automation of complete business processes
- **Remote MCP transport** - Streamable HTTP endpoint for easy integration with any MCP client
