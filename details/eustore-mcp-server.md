# eustore.dev MCP Server

Autonomous sovereign infrastructure for AI agents — S3-compatible object storage and semantic vector database, fully API-driven with no dashboard or human signup required.

## Features

- **Object Storage** — S3-compatible, any S3 client works. EU-hosted (Germany/Finland).
- **Vector Database** — Semantic search, agent memory, RAG collections.
- **Autonomous Billing** — Pay with USDC on Base (<$0.01 gas), Polygon, Arbitrum, or card via Stripe.
- **No Human Required** — Register, provision, store, search, pay — all via API.
- **GDPR Compliant** — EU data residency by design.

## Tools

| Tool | Description |
|------|-------------|
| `register` | Create account (instant, no human needed) |
| `create_bucket` | Create S3 storage bucket in EU |
| `list_buckets` | List all buckets |
| `get_bucket_credentials` | Get S3 access keys |
| `create_collection` | Create vector collection |
| `list_collections` | List collections |
| `insert_vectors` | Store vectors with metadata |
| `search_vectors` | Semantic similarity search |
| `collection_info` | Collection stats |
| `check_balance` | Credit balance |
| `topup_crypto` | Pay with USDC on Base |
| `get_pricing` | Current pricing |
