# Octav API MCP Server

[Source Code](https://github.com/Octav-Labs/octav-api-mcp)

Octav API MCP Server is a multi-chain crypto portfolio tracking MCP server that enables AI assistants to access wallet holdings, DeFi protocol positions, transaction history, and token analytics across 20+ blockchains. It integrates directly with Claude and other MCP-compatible clients.

## Features

### Multi-Chain Portfolio Tracking
- **Wallet Holdings**: Query token balances and portfolio values across 20+ blockchain networks.
- **DeFi Protocol Positions**: Access positions in DeFi protocols including liquidity pools, lending, staking, and yield farming.
- **Transaction History**: Retrieve detailed transaction history for any wallet address.
- **Token Analytics**: Access price data, market cap, volume, and token metadata.

### Supported Chains
- Ethereum, Polygon, Arbitrum, Optimism, Base, BNB Chain, Avalanche, Solana, and 12+ additional networks.

### Use Cases
- Portfolio analysis and tracking via natural language queries in Claude.
- DeFi position monitoring and performance analysis.
- Cross-chain asset discovery and valuation.
- Transaction history research and reporting.

### Integration
- Compatible with Claude Desktop and other MCP-compatible clients.
- Connect via the Octav API using your API key.
