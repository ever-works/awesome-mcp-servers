# PageBolt MCP

**Category:** Media Processing MCP Servers
**Tags:** mcp, screenshot, pdf, video-recording, browser-automation, ai-voice, web-capture

## Description
PageBolt MCP is a native MCP server for screenshots, PDFs, narrated video recording with AI voice, and page inspection. It enables AI agents in Claude Desktop, Cursor, and Windsurf to capture web pages, generate PDFs, record narrated product demos with AI voice narration, and inspect page elements — all through direct tool use without writing code.

## Features
- **Screenshots:** Capture any URL as PNG, JPEG, or WebP with device emulation, ad blocking, and cookie banner removal.
- **PDF Generation:** Convert any URL or HTML to PDF with full-page rendering.
- **Narrated Video Recording:** Record multi-step browser automation workflows as MP4 with AI voice narration synced to each step.
- **Page Inspection:** Get structured maps of page elements with CSS selectors for AI-driven automation.
- **Browser Automation:** Multi-step sequences — navigate, click, fill, scroll — with screenshot and PDF outputs.
- **Device Emulation:** 25+ device presets including iPhone, iPad, MacBook, and more.
- **MCP Native:** Works directly in Claude Desktop, Cursor, and Windsurf via tool use.
